地推团队

岗位职责Job Description

  • 1.根据公司、部门整体规格，制定市场拓展计划.
  • 2.负责对业务工作进行分析、管理和实施.
  • 3.对所管辖区域运营和人员管理包括：覆盖率、活动执行、单量、客诉率，人员稳定性等.
  • 4.负责对各区域的工作指引、培训及问题及时解决.
  • 5.根据部门业务计划，及时制作培训课件，能系统培训业务相关知识，做好落地执行.

任职要求Qualifications

  • 1.有策划经验，线下相关业务活动方案的制定与推广.
  • 2.认真分析市场竞争情况，为经营管理出谋划策，不断扩大市场占有率.
  • 3.良好的线上管理能力，如微信群等.
  • 4.按时完成公司交办的其他工作.

校园总代理

岗位职责Job Description

  • 1.负责招募店长，协助公司管理自己的大学.
  • 2.负责自己所在楼层（包括但不限于）三十九度零食店的运营推广.
  • 3.负责管理自己的校园店长并处理订货订单.

任职要求Qualifications

  • 1.工作认真、谨慎、细心、责任心强.
  • 2.具有较强的沟通、协调能力，优秀的外联、公关能力，具有解决突发事件能力.
  • 3.较强的业务拓展能力，有较好的搜集分析销售线索的能力.
  • 4.有一点管理经验.

校园店长

岗位职责Job Description

  • 1.负责三十九度的信息宣传.
  • 2.负责三十九度的货物处理，订单配送.
  • 3.制定计划并做分析，做好运营推广工作；首次订货由公司免费提供.

任职要求Qualifications

  • 1.在校大学生，男女不限.
  • 2.有较强的沟通能力、亲和力，思维敏捷.
  • 3.工作认真、细致、严谨、条理性强.
  • 4.有较强的组织能力对工作有责任心.
  • 5.有较强的人际交往能力，在院校内有一定的人脉关系.
  • 6.有时间观念，能按照公司要求定时开展宣传活动并及时汇报工作总结.
